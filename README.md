# 🧠 MorphoMinds  
### AI-Powered Reading & Writing Learning Platform for Dyslexia Support

MorphoMinds is a full-stack, AI-driven educational platform designed to help learners — especially individuals with dyslexia — improve reading and writing skills through interactive learning and real-time ML-based feedback.

The system integrates multiple backend services, a modern React frontend, and Machine Learning inference servers to create a modular and scalable learning ecosystem.

---

# 🌟 Core Features

## 📖 Reading Modules
- Hindi Reading (Vowels, Consonants, Words, Sentences)
- English Reading Levels
- Pronunciation Assistance API
- Level-based progression with unlock mechanism
- Interactive flashcards with visual + audio feedback

## ✍️ Writing Modules
- Letter tracing with stroke validation
- Real-time color feedback:
  - 🟢 Green → Correct stroke
  - 🔴 Red → Incorrect stroke
- Advanced writing without dotted guides
- ML-powered handwriting evaluation

## 🤖 AI & ML Integration
- Mirror-letter confusion detection (b/d/p/q)
- Stroke pattern recognition
- Model-based letter validation
- Modular ML inference server (FastAPI + TensorFlow)
